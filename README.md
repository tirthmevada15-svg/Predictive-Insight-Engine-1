# 🏠 House Price Prediction Project

## 📌 Overview

This project builds a Machine Learning model to predict house prices
using multiple features.

## 📊 Dataset Features

-   area_sqft
-   bedrooms
-   bathrooms
-   location_score
-   age_years
-   distance_city_km
-   lot_size_sqft
-   has_garage
-   has_pool
-   renovation_years_ago
-   house_price_inr (target)

## 📈 Models Used

-   Simple Linear Regression (SLR)
-   Multiple Linear Regression (MLR)
-   Polynomial Regression

## 📊 Model Performance

-   SLR: \~0.56
-   MLR: \~0.92
-   Polynomial: \~0.97

## 📉 Assumptions

-   Residuals randomly distributed
-   Normal distribution (Q-Q plot)
-   Bell-shaped histogram

## ⚙️ Gradient Descent

-   Loss decreases over iterations
-   Model converges properly

## 📌 Key Insights

-   Area strongly affects price
-   Distance reduces price
-   Polynomial gives best accuracy

## 🛠️ Tech Stack

-   Python
-   NumPy, Pandas
-   Matplotlib, Seaborn
-   Scikit-learn
